# Chargeback Risk Dashboard

## Overview

This module focuses on identifying merchants with elevated chargeback risk using transaction and dispute data. The results are designed to be visualized in Power BI for monitoring and decision-making.

## What This Does

- Calculates chargeback rates per merchant using SQL
- Flags merchants exceeding acceptable risk thresholds
- Supports visualization with conditional formatting in Power BI

## Tools & Technologies

- SQL
- Power BI
- CSV datasets

## Outcome

The dashboard enables quick identification of high-risk merchants, supporting fraud mitigation and risk management efforts.
