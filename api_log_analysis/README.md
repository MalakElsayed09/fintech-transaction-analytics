# Payment Gateway API Log Analysis

## Overview

This module analyzes API logs from a payment gateway to identify common causes of failed transactions and measure overall system reliability.

## What This Does

- Parses structured API log data
- Identifies authentication errors, server failures, and timeouts
- Calculates system uptime based on successful responses
- Summarizes operational issues for troubleshooting

## Tools & Technologies

- Python
- pandas
- JSON log parsing

## Outcome

The analysis highlights key failure patterns and helps guide corrective actions to maintain high system uptime for merchants.
