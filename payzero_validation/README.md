# PayZero Algorithm Validation

## Overview

This module validates the PayZero cash discount pricing algorithm to ensure accurate card pricing calculations across a variety of transaction scenarios.

## What This Does

- Loads predefined test cases from a CSV file
- Applies the PayZero pricing formula
- Compares calculated prices against expected values
- Automatically flags any failed test cases

## Tools & Technologies

- Python
- pandas
- Automated test validation

## Outcome

All test cases pass successfully, confirming correct pricing logic and helping prevent calculation errors before deployment.
